## テンプレート既定用語

外部認証基盤 ap-safedx 連携で参照する既定エンティティ。命名は変更しない。

| 日本語名 | 英語名 | コード上の命名 | 定義 |
|---|---|---|---|
| システムユーザー | System User | `MSystemUser` | 外部認証基盤で認証された利用者。外部DB参照。`user`・`customer` は使わない |
| 部署 | Department | `MDepartment` / `m_department` | 外部認証基盤の部署マスタ。読み取り専用 |
| 部署メンバー | Department Member | `MDepartmentMember` / `m_department_member` | アカウントと部署の紐づけ。読み取り専用 |
| アカウントID | Account ID | `account_id` | 利用者の一意識別子。M365 または SYSTEM 種別 |
| アプリ設定 | App Setting | `MAppSetting` / `m_app_settings` | キー・バリュー形式のアプリ設定 |
| 論理削除 | Logical Deletion | `deleted` | 物理削除せず削除フラグで無効化する方式 |
| セッション | Session | `session` | Redis 上に保持する認証状態。TTL は SESSION_TTL_HOURS |

## 命名規約（既定）

| 種別 | プレフィックス | 例 |
|---|---|---|
| マスタテーブル | `m_` | `m_example_categories` |
| トランザクションテーブル | `t_` | `t_example_items` |
| 外部参照テーブル | なし（ExternalBase 継承） | `m_department` |
