## リストア手順

1. 対象日時のバックアップファイルを特定する
2. 対象環境の App DB を停止または退避する
3. バックアップから App DB をリストアする
4. Alembic のリビジョンとアプリのモデル定義が整合するか確認する
5. ヘルスチェック `{{APP_ROOT_PATH}}/health` と主要機能で復旧を確認する
