ペルソナ・ユーザーストーリー

{{アプリ名}} の利用者像と、その利用者が求める行動を定義する。

記入ガイド

ペルソナを先に固めてからユーザーストーリーの主語を埋める。

ストーリーは「〜として〜したい。〜のため」の形式を厳守する。

MVP対象の○が最初のリリーススライス。優先度は 高/中/低。

ペルソナは困りごと起点でニーズを導く。属性は行動に影響する範囲で書く。

ストーリーの ID は US-連番。ペルソナ列にペルソナIDを対応させる。優先度と MVP対象は開発順序の根拠になる。

リリーススライスは MVP対象=○ を第1リリースに割り当てる。後続は次リリース以降。
