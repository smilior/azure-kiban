用語集（ユビキタス言語）

{{アプリ名}} のドメイン用語を統一する。日本語名・英語名・コード上の命名・定義を1対1で対応させる。

記入ガイド

用語の揺れはそのままAIのコードの命名の揺れになる。実装開始前に必ず全ドメイン用語を埋める。

アプリ固有用語の表は記入例。1行1用語で自アプリの全ドメイン用語を追加する。

テンプレート既定用語

外部認証基盤 ap-safedx 連携で参照する既定エンティティ。命名は変更しない。
